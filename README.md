# CrudEscuela

Sistema web escolar que permite la asignación de division en las areas de edificio y especialidad.

Que contiene el sistema:
*Manejo de operaciones asincronas
*Relacion entre tablas usando llaves primarias y foraneas
*Arquitectura N-Capas

Por mejorar:
*Manejar el nombre de la division en lugar del IDDivision en las tablas de edificio y especialidad.
*Ocultar los botones para recargar la tabla que muestra la información
*Cambiar los textboxt que muestran el estado de la operación por un mensaje emergente.
